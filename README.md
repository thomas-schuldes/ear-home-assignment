# EAR – Home Assignment (Mengenmeldesystem)

Dieses Repository enthält die Lösung für das EAR-Home-Assignment zur Entwicklung eines Systems zur Erfassung und Übermittlung von Mengenmeldungen.

## Ziel

- Web-Frontend zur Erfassung und Anzeige von Mengenmeldungen
- Java-Backend mit REST-API, Persistenz und Anbindung an einen bestehenden SOAP-Service
- Dokumentation der Architektur, Tickets und Architekturentscheidungen

## Projektstruktur

```text
ear-home-assignment/
  backend/        # Spring Boot Backend (Java, REST, Persistenz, SOAP-Anbindung)
  frontend/       # Einfaches Web-Frontend (HTML/JS)
